# Tic Tac Toe Game

# Overview

This project is a simple implementation of the Tic Tac Toe game using Python.
It includes both a terminal version and a graphical user interface (GUI) version.
The game allows a player to play against another player or against the computer.

---

## Features

  . Two game modes

  . Human vs Human
  . Human vs Computer

#  Difficulty levels (in AI mode)

  . Easy
  . Hard

. Score tracking system

. Keeps track of wins for each player

. Replay option

. Restart the game without closing the program

 # Graphical interface

  . Interactive board using GUI
  . Clear display of moves and results



## How It Works

. The game board is a 3x3 grid
. Players take turns marking X and O
. The first player to align three marks horizontally, vertically, or diagonally wins
 If all cells are filled and no player wins, the game ends in a draw

In AI mode:

. Easy level makes random moves
. Hard level uses logical decision-making to play optimally
