# CODESOFT AI TASKS

## Overview

This repository contains all the tasks completed as part of the AI internship.
Each task is organized into separate folders with its own implementation and documentation.

The goal of these projects is to understand basic concepts of artificial intelligence, logic building, and problem-solving using Python.

---

## Projects Included

* Chatbot System
  A simple chatbot that responds to user queries using rule-based logic and similarity matching.
  It also includes a GUI version for better interaction.

* Tic Tac Toe Game
  A classic game implemented in both terminal and GUI versions.
  It includes features like difficulty levels, score tracking, and replay option.

* Recommendation System
  A book recommendation system that suggests books based on user preferences such as genre, author, and book similarity.
  It uses a content-based filtering approach.

---

## Folder Structure

* chatbot
  Contains chatbot implementation and GUI version

* tic_tac_toe
  Contains terminal and GUI versions of the game

* recommender
  Contains book recommendation system and ML-based version

---

## Technologies Used

* Python
* Tkinter (for GUI applications)
* Basic data structures and logic

---

## How to Run

1. Open terminal in the project folder
2. Navigate to the specific project folder

Example:

```bash
cd chatbot
python chatbot.py
```

Similarly, you can run other projects from their respective folders.
